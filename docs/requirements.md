# Job Application Tracker

## Project Goal

The purpose of this application is to help users track all their job applications in one place.

Users should be able to manage applications, monitor their status, and keep notes related to interviews and hiring processes.

---

## Version 1.0 Scope

### Authentication

* Register Account
* Login Account

### Job Application Management

* Add Job Application
* View All Applications
* View Single Application
* Update Application
* Delete Application

### Search

* Search by Company Name
* Search by Job Role
* Search by Status

---

## Job Application Fields

* Company Name
* Job Role
* Application Date
* Status
* Notes

---

## Status Values

* Applied
* Interview Scheduled
* Interview Completed
* Rejected
* Selected

---

## Out Of Scope (Future Versions)

* Email Notifications
* Resume Upload
* Interview Reminders
* Analytics Dashboard
* Admin Panel
